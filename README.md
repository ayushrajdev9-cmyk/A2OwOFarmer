# A2 OWO FARMER

Ultimate OwO Bot Farming Suite

**Made by Ayush Rajdev & Anzar Iqbal**

## Features

- **Auto Farm** - Hunt, sell, flip, cash loop automatically
- **Blackjack Farm** - Smart blackjack gambling with basic strategy
- **Ban Bypass** - Built-in safety features
- **Real-time Stats** - Track wins, losses, profits
- **Timer & Stop-Loss** - Set limits for safe farming

## Installation

```bash
pip install discord.py-self colorama
```

## Usage

```bash
python a2.py YOUR_DISCORD_TOKEN
```

Or run without args to enter token interactively.

## Commands

| Command | Description |
|---------|-------------|
| `.autofarm` | Start auto hunt/sell/flip/cash loop |
| `.stopfarm` | Stop auto farm |
| `.bj` | Start blackjack farming |
| `.bjstop` | Stop blackjack |
| `.bjtimer 30m` | Set auto-stop timer |
| `.bjstoponloss 500k` | Set loss limit |
| `.bjbets Low/High` | Change bet sequence |
| `.bjstatus` | Show stats |
| `.help` | Show help |
